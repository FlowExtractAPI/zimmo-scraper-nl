# zimmo-scraper
Plak gewoon een zimmo.be zoek-URL rechtstreeks uit je browser. De scraper haalt de volledige detailpagina op voor elke advertentie en slaat de ruwe API-respons op — zo krijg je elk veld dat zimmo.be beschikbaar stelt: prijs, EPC-certificaten, overstromingsstatus, ruimtelijke ordening, kadastraal inkomen, agentschapsgegevens, afbeeldingen
